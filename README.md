 
# 🐍 Python: Do Zero ao Mestre

Repositório dedicado ao estudo da linguagem **Python**, com foco em lógica de programação, manipulação de dados e automação. Este espaço documenta a minha transição técnica e a expansão do meu toolkit de desenvolvimento.

## 🎯 Objetivos do Repositório

* Consolidar fundamentos da linguagem (sintaxe, Estruturas de Dados, OOP).
* Resolver desafios de lógica e algoritmos aplicados.
* Explorar bibliotecas de Análise de Dados e Automação.
* Explorar o processamento de dados e integração com serviço cloud.
* Documentar a evolução de conceitos de Backend para o ecossistema de dados.

---

## 📂 Estrutura de Aprendizado

O repositório está organizado por módulos progressivos:

* **`01-fundamentos/`**: Variáveis, tipos dinâmicos, estruturas condicionais e de repetição.
* **`02-estruturas-dados/`**: Listas, Dicionários, Tuples, Sets e manipulação de Strings.
* **`03-funcoes-modulos/:`** Funções, Lambda, List Comprehension e Importações.
* **`04-orientacao-objetos/`**: Classes, métodos estáticos, herança e interfaces.
* **`05-tratamento-erros/:`** Try/Except e Logging.
* **`05-data-handling/`**: Manipulação de ficheiros CSV, JSON e introdução a bibliotecas de dados.
* **`06-automacao-e-apis/`**: Scripts de automação e consumo de APIs REST.
* **`07-desafios-praticos/:`** Projetos pequenos (Calculadoras, Scraping, Automação).

## 🛠️ Stack Técnica

* **Linguagem:** Python 3.12+
* **Ambiente:** VS Code / PyCharm
* **Gestão de Dependências:** `pip` / `venv`
* **Cloud Context:** Exercícios focados em portabilidade para Azure e GCP.

## 🚀 Como Executar

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/teu-utilizador/estudos-python.git
    ```
2.  **Crie um ambiente virtual (opcional, mas recomendado):**
    ```bash
    python -m venv venv
    source venv/bin/activate  # Linux/Mac
    # venv\Scripts\activate  # Windows
    ```
3.  **Execute um script de exemplo:**
    ```bash
    python 01-fundamentos/exercicio_exemplo.py
    ```
